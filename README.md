
# archiscribe-corpus

This is the corpus repository for https://archiscribe.jbaiter.de.

The goal is to have as much diverse OCR ground truth for 19th Century German
prints as possible.

Currently the corpus contains 4255 lines from 112 works
published across 73 years. Detailed statistics are available below.
